# OIBSIP_domain_taskno.5
Sales Prediction Using Python is a machine learning project that predicts future product sales based on factors such as advertising expenditure, target audience segment, and marketing platform. This project demonstrates the practical use of regression techniques in business analytics.

Sales prediction plays an important role in business planning and decision-making. This project uses Python and machine learning techniques to predict product sales based on advertising spend, marketing strategy, and other influencing factors. The model helps businesses optimize advertising costs and improve sales performance.

## what my project does??
- Analyze sales and advertising data
- Understand the relationship between advertising and sales
- Build a machine learning regression model
- Predict future sales
- Evaluate model performance

## Dataset used usually contains historical sales data
- Features may include:
  - Advertising budget
  - Marketing platform
  - Target audience segment
  - Sales values
